# Quick Start (5 Minutes)

Get up and running in 5 minutes if you already have Supabase and Vercel accounts.

## 1. Install (30 seconds)
```bash
cd microsite-gen
npm install
```

## 2. Setup Supabase (3 minutes)

**Create project** → **Run SQL** (`supabase-schema.sql`) → **Create bucket** (`business-files`, public)

Get credentials from Settings → API.

## 3. Configure (30 seconds)
```bash
cp env.template .env.local
# Edit .env.local with your Supabase credentials
```

## 4. Run (10 seconds)
```bash
npm run dev
```

Visit http://localhost:3000

## 5. Test (1 minute)
Upload a text file with business info. Done!
